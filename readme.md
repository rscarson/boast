This is the repository for the BIOD paper (How Many Iterations are Enough? Bayesian Iterative Outlier Detection for Conservative Randomized Testing)

## Structure

- 'examples' contains a sample using the rust impl of the algorithm
- 'latex-src' is the latex source files for the pdf
- 'rust_impl' contains a full featured rust crate for the technique
- 'bayesian_iterative_outlier_detection.draft.pdf' is the most recent version of the paper
- 'biod.py' is a python example of the technique
- 'build_latex.bat' can be used to recompile the paper
- 'fit_ps.py' is used to find a new p_s constant value if C is altered
